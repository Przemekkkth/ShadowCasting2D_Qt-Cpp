Simple visual representation of shadow casting in 2D space.

### Installing
A step by step series  that tell you how to get a execute project.
Get it from GitHub
```
git clone git@github.com:Przemekkkth/ShadowCasting2D_Qt-Cpp.git
```
Compile
```
qmake && make
```
